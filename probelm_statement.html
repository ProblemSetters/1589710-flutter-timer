Create a Timer application as shown below per the given requirements.  The finished application must pass all of the unit tests.

 

 

Complete the implementation of lib/timer_page.dart according to the following requirements:

 

  • The app consists of one screen:
    • In the view
      • display the Initial value of the timer
      • display the Start button
      • display the edit button on top of the screen
    • Clicking on the start button, the timer should start decreasing.
      •  the timer should only stop at 0.

 

Complete the implementation of lib/timer_controller.dart according to the following requirements:

 

  • Fill in the _timer code to periodically decrease the value by 1 in every 1 second and stop at 0

 

Initially, the file is filled with boilerplate code. Note the following:

  • The "Initial Value of Timer" must be "10".
  • The "Value of Timer" must use "Text" Widget .
  • The "start button" must use "ElevatedButton" Widget .
  • The "start button" must be named "Start Counter" .

 

The component has these key attributes for test cases and certain classes for rendering purposes. They should not be changed.
